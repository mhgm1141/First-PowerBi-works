# First PowerBi works
Interactive Power BI dashboard analyzing salaries, tools, and job satisfaction of 630 data professionals.

Good day :)
I’ve been practicing Power BI extensively because I’m highly interested in data analytics. This is my first project published on GitHub.
They say a picture is worth a thousand words, so I decided to document it and share my progress.

This project was inspired by a course on Udemy. While the dataset and general concepts come from the course, the dashboard development, visual design, and data storytelling were created independently using Power BI.  
All credit for the original dataset and learning materials goes to the course instructor.

--------------------------------------------

Overview

This project is an interactive dashboard built in Power BI that analyzes the results of a survey conducted with 630 data professionals worldwide.
The goal is to provide insights into the current landscape of the data industry, including roles, salaries, tools, and job satisfaction.

--------------------------------------------

Key Questions Answered

- How much does each data role earn on average?  
- How difficult is it to break into the data industry?  
- Which programming languages are most used by role?  
- How satisfied are professionals with their salary and work-life balance?  
- Which countries have the highest number of respondents?  

--------------------------------------------

General Overview

- Total respondents: 630  
- Average age: 29.87 years  
- Work-life balance satisfaction: 5.74 / 10  
- Salary satisfaction: 4.27 / 10
